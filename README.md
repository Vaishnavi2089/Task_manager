# 📝 Task Manager (Python - Console Based)

A simple **Task Manager application** built using **Python** that performs basic **CRUD operations** on tasks using a console interface.

---

## 🚀 Features

* ➕ Add new tasks
* 📋 View all tasks
* ✏️ Update tasks (mark as completed / edit title)
* 🗑️ Delete tasks
* 🎯 Menu-driven console interface

---

## 🛠️ Tech Stack

* Python
* OOP Concepts
* Lists (for data storage)

---

## 📂 Project Structure

```
Task_manager/
│
├── main.py              # Main program (console UI)
├── task.py              # Task class
├── task_manager.py      # Core logic (CRUD operations)
└── .gitignore
```

---

## ▶️ How to Run

### 1. Clone the repository

```
git clone https://github.com/Vaishnavi2089/Task_manager.git
cd Task_manager
```

### 2. Run the application

```
python main.py
```

---

## 🧠 How It Works

* Tasks are stored in a list during runtime
* Each task has:

  * ID
  * Title
  * Completion status
* User interacts through a menu system
